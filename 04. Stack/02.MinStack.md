# 155. Min Stack
Easy


Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

Implement the MinStack class:

- MinStack() initializes the stack object.
- void push(int val) pushes the element val onto the stack.
- void pop() removes the element on the top of the stack.
- int top() gets the top element of the stack.
- int getMin() retrieves the minimum element in the stack.
 

Example 1:
```
Input
["MinStack","push","push","push","getMin","pop","top","getMin"]
[[],[-2],[0],[-3],[],[],[],[]]

Output
[null,null,null,null,-3,null,0,-2]

Explanation
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin(); // return -3
minStack.pop();
minStack.top();    // return 0
minStack.getMin(); // return -2
 ```

Constraints:

- -231 <= val <= 231 - 1
- Methods pop, top and getMin operations will always be called on non-empty stacks.
- At most 3 * 104 calls will be made to push, pop, top, and getMin.

```Approach
Two Common Approaches
1) Two Stacks (st + minSt)

Invariant: minSt.peek() is the minimum of all elements currently in st.
Push:

Push val onto st.
If val <= minSt.peek(), also push val onto minSt.


Pop:

If st.peek() == minSt.peek(), pop from minSt too.
Pop from st.


GetMin: Return minSt.peek().

Why it works:
Every time the global minimum changes (or ties), you record it on minSt. Syncing pops keeps the minimum correct.
Trade-off:
Simple, but you manage two structures.

2) Single Stack of Pairs ([value, currentMin])

Invariant: For each element, store (value, minUpToThisPosition).
Push:
currentMin = st.isEmpty() ? val : min(val, st.peek().min)
Push [val, currentMin].
Pop: Pop once.
GetMin: Return st.peek().min.

Why it works:
The minimum at any time is exactly the min stored with the top elementâ€”no extra stack needed.
Trade-off:
Uses a bit more per-element storage (a pair), but is simpler to reason about and avoids syncing logic.
```

```java

class MinStack {
    Stack<int[]> st;

    public MinStack() {
        st = new Stack<>();
    }

    public void push(int val) {
        if (st.isEmpty()) {
            st.push(new int[]{val, val});
        } else {
            int currentMin = Math.min(val, st.peek()[1]);
            st.push(new int[]{val, currentMin});
        }
    }

    public void pop() {
        st.pop();
    }

    public int top() {
        return st.isEmpty() ? -1 : st.peek()[0];
    }

    public int getMin() {
        return st.isEmpty() ? -1 : st.peek()[1];
    }
}
```
