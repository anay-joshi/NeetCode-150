## ðŸ” Intuition

Inverting a binary tree means **swapping the left and right child of every node**.

Since each subtree is itself a binary tree, the same logic can be applied recursively to all nodes.

---

## ðŸ§  Approach 1: Recursive (DFS)

### Steps
1. If the current node is `null`, return `null`
2. Swap the left and right children
3. Recursively invert the left subtree
4. Recursively invert the right subtree
5. Return the root

### Complexity
- **Time:** `O(n)` â€” every node is visited once
- **Space:** `O(h)` â€” recursion stack (`h` = height of tree)

### Java Implementation
```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        if (root == null) {
            return null;
        }

        // Swap left and right children
        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;

        // Recursively invert subtrees
        invertTree(root.left);
        invertTree(root.right);

        return root;
    }
}
